- `THANKS_REACT_TO_LAST_MESSAGE_SLACK_WEBHOOK` and 4 other react webhooks
- `LOW_ON_BEANS_WEBHOOK` (or whatever the actual key is — check the file)
- `BREWING_WEBHOOK`, `READY_WEBHOOK` (check actual names)
- `GOOGLE_SHEET_URL`
- `WEATHER_LAT`, `WEATHER_LON`

## Files to touch

## Specification
Mirror every key currently in `settings.toml` with empty / placeholder values. Keep the keys for v1 features that v2 will remove (weather, react webhooks) — Phase 5 (P5.2 / P5.3) will trim them. This task is a snapshot; cleanup is later.

Use this format:

```toml
CIRCUITPY_WIFI_SSID = ""
CIRCUITPY_WIFI_PASSWORD = ""

SEND_MESSAGE_SLACK_WEBHOOK = ""
# ... mirror remaining webhook keys with empty strings

GOOGLE_SHEET_URL = ""

# Person responsible for refilling beans; substituted into the
# Low-on-beans success message. Defaults to "Someone" if unset.
LOW_BEANS_PERSON = ""

WEATHER_LAT = ""
WEATHER_LON = ""
```

`LOW_BEANS_PERSON` is **new in v2** — add it even though the v1 `settings.toml` does not have it.
