## Specification
Rewrite `CLAUDE.md` end-to-end. Keep it tight (target ≤ 150 lines vs the current ≈ 75). Cover:

1. **What this is** — Slack-connected coffee gadget on Waveshare ESP32-S3 Touch LCD 2; CircuitPython 9.2.8.
2. **Deploying / running** — same as v1: USB drive `CIRCUITPY`, `code.py` is the entry point, `settings.toml` for env vars, `lib/` ships with the source. Mention `deploy.sh` for the macOS rsync watcher.

4. **Things to know before editing**:
   - All hit-tests go through `utils.touch.normalize`. Never inline the rotation.
   - Touch dispatch is edge-triggered via `utils.touch.TouchTracker`. Don't poll `ctp.touches` directly.
   - Visual feedback uses non-blocking timestamps (`card.flash(now)` + `card.tick(now)`). **No `time.sleep` for UI feedback.**
   - All HTTP goes through one injected `SendRequest` instance. Methods return `(ok, body)`.
   - Screens implement the Navigator protocol (`get_group`, `on_enter`, `handle_touch`, `tick`).
   - Settings (`settings.toml`) are gitignored; `settings.example.toml` is the template.
5. **Known gaps / deferred features** — short list mirroring `V2-ARCHITECTURE.md` §11 "Deferred" (WifiIndicator placement, stats screen, milestone messages, react/emoji flow).
