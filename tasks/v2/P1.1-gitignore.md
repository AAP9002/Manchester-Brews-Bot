# P1.1 — Add .gitignore

**Status:** done
**Phase:** P1 — Foundation
**Depends on:** none
**Blocks:** —
**HLD reference:** §9 Configuration & secrets

## Goal
Stop live secrets from being committable.

## Context
The repo has no `.gitignore` and no commits yet. `settings.toml` contains real WiFi credentials and Slack webhook URLs. CircuitPython also writes `boot_out.txt` on boot, and the SD card mount has runtime files we don't want tracked.

## Files to touch
- `.gitignore` (create at project root)

## Specification
Create `.gitignore` with at minimum:

```
# CircuitPython runtime
boot_out.txt
__pycache__/
*.mpy.tmp

# Local secrets
settings.toml

# Editor / OS
.DS_Store
.idea/
.vscode/
```

Keep `lib/` tracked (it holds the bundled Adafruit `.mpy` modules that ship with the firmware).

Keep `sd/placeholder.txt` tracked but ignore other contents of `sd/` if any appear:

```
sd/*
!sd/placeholder.txt
```

## Acceptance criteria
- [ ] `.gitignore` exists at the project root.
- [ ] `git status` no longer lists `settings.toml` as untracked.
- [ ] `lib/` and `sd/placeholder.txt` are still tracked / trackable.

## Out of scope
- Do not delete or modify `settings.toml`.
- Do not commit anything.
- Do not rewrite history.
