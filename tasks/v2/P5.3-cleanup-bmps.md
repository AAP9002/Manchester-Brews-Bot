# P5.3 — Remove unused BMPs and react webhook config

**Status:** not-started
**Phase:** P5 — Cleanup
**Depends on:** P4.5
**Blocks:** P5.4
**HLD reference:** §10 File structure

## Goal
Trim the asset and config debt left behind by the removed react/emoji feature.

## Context
v1 had a 7-emoji `ReactScreen` that we are not bringing back in v2. The associated BMPs and webhook env-var entries are no longer referenced after P5.1.

## Files to touch
- `images/` — delete unused BMPs
- `settings.example.toml` — remove react webhook keys

## Specification

1. **BMPs to delete** (verify by listing `images/` and `grep`-ing each filename across the repo first):
   - `images/thanks.bmp`
   - `images/coffee-parrot.bmp`
   - `images/coffee-blob.bmp`
   - `images/thanks-dog.bmp`
   - `images/rachel.bmp`
   - `images/lets-go.bmp`
   - `images/star.bmp`
   - `images/react-coming-soon.bmp`
   - `images/react.bmp`

2. **BMPs to keep**:
   - `images/back.bmp`, `images/loading.bmp`, `images/coffee-done.bmp` (still used by v2 screens)
   - Any new v2 card icons that have shipped (`announcement.bmp`, `low-beans.bmp`, `log-intake.bmp`)

3. **`settings.example.toml` keys to remove**:
   - `THANKS_REACT_TO_LAST_MESSAGE_SLACK_WEBHOOK`
   - The 4 other react webhook keys (parrot, blob, thanks-dog, rachel, lets-go, star — confirm names against the file)

4. **`settings.example.toml` keys to keep**:
   - `CIRCUITPY_WIFI_SSID`, `CIRCUITPY_WIFI_PASSWORD`
   - `SEND_MESSAGE_SLACK_WEBHOOK`, `BREWING_WEBHOOK`, `READY_WEBHOOK`, `LOW_ON_BEANS_WEBHOOK` (4 webhooks total)
   - `GOOGLE_SHEET_URL`

5. Verify nothing else in the repo references the removed assets:
   ```
   grep -rn "thanks.bmp\|coffee-parrot\|coffee-blob\|thanks-dog\|rachel\|lets-go\|star.bmp\|react-coming-soon\|react.bmp" .
   ```
   Hits in `tasks/` or docs are fine; hits in `screens/`, `components/`, `utils/`, `code.py` are not.

## Acceptance criteria
- [ ] All listed BMPs deleted from `images/`.
- [ ] `settings.example.toml` has only the 4 webhook keys + WiFi + Google Sheet (and any non-webhook keys still in use).
- [ ] `grep` confirms no code references to the removed BMPs.
- [ ] **On-device verification: device boots and every v2 flow still works.**

## Out of scope
- Do NOT delete the user's local `settings.toml` keys unless asked — those are inert.
- Do NOT add a `THANKS_*` placeholder back; if react/emoji ever returns it'll be a fresh design.
