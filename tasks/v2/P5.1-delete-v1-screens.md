**Status:** not-started
**Phase:** P5 — Cleanup
**Depends on:** P4.5 (v2 boots into HomeScreen)
**Blocks:** P5.4

## Goal
Remove the v1 screen files now that v2 has shipped.

## Context
After P4.5, `code.py` no longer imports or instantiates `MenuScreen`, `BroadcastScreen`, or `ReactScreen`. The files remain only as historical baggage. `ReactScreen` was already commented out in v1; `MenuScreen` and `BroadcastScreen` were the v1 home and announcement screens.

## Files to touch
- `screens/MenuScreen.py` (delete)
- `screens/BroadcastScreen.py` (delete)
- `screens/ReactScreen.py` (delete)

## Specification
1. Confirm with `grep -r "MenuScreen\|BroadcastScreen\|ReactScreen" .` that the only remaining references are within these three files themselves.
2. `git rm` (or simply remove) the three files.

## Acceptance criteria
- [ ] `screens/MenuScreen.py`, `screens/BroadcastScreen.py`, `screens/ReactScreen.py` no longer exist.
- [ ] `grep -r "MenuScreen\|BroadcastScreen\|ReactScreen" .` returns no hits outside markdown / docs / tasks dirs.
- [ ] **On-device verification: device still boots into the v2 home screen and every flow works.**

## Out of scope
- Do NOT delete `screens/HomeScreen.py`, `screens/AnnouncementScreen.py`, `screens/SuccessScreen.py`, `screens/NoConnectionScreen.py`.
- Do NOT touch the `screens/__init__.py` if one exists — leave directory infrastructure untouched.
- Do NOT remove BMPs here — that's P5.3.
